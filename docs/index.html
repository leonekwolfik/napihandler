NAPIHandler

Lekki handler API dla Node.js — obsługa routingu, walidacja i konfiguracja

Opis projektu

NAPIHandler to prosty, modułowy handler do budowy API w Node.js. Ułatwia tworzenie endpointów, walidację danych i obsługę błędów.

Funkcje

  • Prosty routing i mapowanie handlerów
  • Walidacja wejścia
  • Obsługa błędów i standaryzowane odpowiedzi
  • Łatwy w integracji z istniejącymi aplikacjami Node

Jak zacząć

  1. Sklonuj repo: git clone https://github.com/leonekwolfik/napihandler.git
  2. Zainstaluj zależności: npm install
  3. Uruchom testowy serwer: node example.js (lub zgodnie z README repo)

Przykład użycia

// przykładowy handler
const NAPI = require('napihandler');

const app = new NAPI();
app.get('/hello', (req, res) => {
  res.json({ message: 'Hello world' });
});

app.listen(3000);
