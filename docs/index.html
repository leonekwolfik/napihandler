open source · cli tool

napihandler

Download subtitles from NapiProjekt via the napiprojekt: protocol on macOS and Linux.

↓ Download latest ⌥ View on GitHub
// how it works
01
Download the binary

Grab the right build for your system from the Releases page — no runtime, no dependencies.

02
Register the protocol

Run napihandler --register once. This hooks into your OS so clicking a napiprojekt: link triggers napihandler automatically.

03
Click any link on napiprojekt.pl

The subtitle file is downloaded to your current directory as a .srt file. Done.

// installation

Get started

# download and install
curl -Lo napihandler https://github.com/leonekwolfik/napihandler/releases/latest/download/napihandler-linux-x86_64
chmod +x napihandler && sudo mv napihandler /usr/local/bin/
# register napiprojekt: protocol (once)
napihandler --register
# download and install (Apple Silicon)
curl -Lo napihandler https://github.com/leonekwolfik/napihandler/releases/latest/download/napihandler-macos-arm64
chmod +x napihandler && sudo mv napihandler /usr/local/bin/
# register napiprojekt: protocol (once)
napihandler --register
// usage

CLI reference

# paste a link directly from napiprojekt.pl
napihandler "napiprojekt:07a1046ccddd59c0ffc7932331a16d63"
# or just the MD5 hash
napihandler 07a1046ccddd59c0ffc7932331a16d63
# different language
napihandler "napiprojekt:HASH" --jezyk EN
# custom output filename
napihandler "napiprojekt:HASH" -o film.srt
